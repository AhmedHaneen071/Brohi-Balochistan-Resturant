# Brohi Balochistan | Bar B.Q & Restaurant

A premium restaurant website for **Brohi Balochistan**, a BBQ & restaurant brand in Balochistan. Built with React, Vite, Tailwind CSS v4, and Framer Motion.

## Tech Stack

- **React 19** — UI library
- **Vite 8** — Build tool and dev server
- **Tailwind CSS v4** — Utility-first CSS framework
- **Framer Motion** — Animation library
- **Lucide React** — Icon set

## Features

- Responsive single-page restaurant website
- Image preloading with animated loading screen
- Smooth scroll navigation
- Menu, features, timing, and testimonials sections
- Dark theme with gold accent color palette
- Custom animations and transitions

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Lint code
npm run lint
```
